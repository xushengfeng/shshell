# shshell

混合终端（模拟器）和 shell，受 warp 等超级终端启发，即图形化 shell

输入

-   [x] 自由编辑，可通过鼠标点击跳转，可选择替换，同在编辑器一样
-   [x] 补全和提示 仅支持最基本的命令和路径补全
-   [ ] 记忆

运行

-   [ ] 管道
-   [ ] 循环、变量等

输出

-   [x] 可回退到 Xterm.js
-   [ ] 转义序列
    -   [ ] CSI
        -   [x] SGR 样式（颜色、下划线等）
        -   [ ] 光标控制
        -   [ ] 键盘输入
        -   [ ] 鼠标输入
-   [ ] 输出限制：单个命令缓冲区限制，命令级别移除，移除错误输出
-   [ ] 压缩输出到内存中
-   [ ] 持久化
-   [ ] 折叠输出

考虑到把传统 shell 的输入输出改成了 dom 元素，可以以输入输出为单位控制渲染，比如可以移除一些无用的错误输出。
